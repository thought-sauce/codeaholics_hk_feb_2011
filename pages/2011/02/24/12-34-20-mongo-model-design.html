Mongo: Embed vs Relational

Embed (de-normalized)


company model:
{
  _id: "897132", name: "Thought Sauce", 
  employee: {_id: "12345", name: "Eddie" }
}

Relational (normalized)


company model:
{
  _id: "897132", name: "Thought Sauce"
}


employee model:
{
  _id: "12345", name: "Eddie", company_id: "897132" 
}
